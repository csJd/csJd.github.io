Windows 上使用 zip 包安装 MySQL

Posted on 2017-03-07 | In Experience

由于重装系统比较频繁，每次都使用安装包安装 MySQL 的话比较麻烦，看到 MySQL 官网提供了 MySQL 的 ZIP Archive，查了下官方文档，这里记录一下使用 ZIP 包安装 MySQL 的方法。
